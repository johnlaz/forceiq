📖 ForceIQ Reference Guide
⚡ Getting Started
1
Configure Settings
Open ⚙️ Settings first. Add your Groq API key (free at console.groq.com). Fill in Company Profile — name, mission, founder name, current phase. Add Voice & Tone — this is the most important field. Without it, outputs are generic.
2
Run the Setup Wizard
Click 🧙 Setup Wizard in the toolbar. It walks through company profile, API key, agents, and platforms in 5 steps. The wizard has AI-assist buttons — describe your company in plain English and it drafts your mission and voice for you. You can also paste a URL and it'll scrape context automatically.
3
Deploy Your First Agents
Go to Hiring Hall → Roster → + Hire Agent. Each agent needs a Role and Goal — the Goal is most important, be specific. Agents are assigned to Departments for smart mail routing and gap analysis. Need a department that isn't in the built-in list? Hiring Hall → Departments → + New Department.
🔀 Running Work
4
Workflows — Templates & Custom
Workflows → Templates has 5 built-in starting points. Workflows → My Workflows shows your custom ones. Workflows → Build lets you create any pipeline — name it, pick an icon, write N steps each with a system prompt. Enable the Critic Loop to have each step self-corrected against your Never Say list before passing output to the next agent.
5
Prompt Builder & Clarity Gate
Before running any workflow, click the 🔧 button next to Deploy. The Prompt Builder shows you the full system context, lets you edit your task, and previews every step. If "Check for ambiguity" is on, the system asks clarifying questions before running — so you never get a full pipeline output based on a misunderstood task.
6
Approval Gates & Deliverables
Custom workflows can require your approval before output files to the Registry. When enabled, output lands in Deliverables → Pending Review. Click the card to read, edit, approve, or send back with a note. Approved outputs go to Registry and Output Vault automatically.
7
Template Chaining
In the Workflow Builder, set "On complete →" to another workflow. When the first finishes, a prompt bar appears offering to launch the next one. Chain an entire operating sequence — Grant Readiness → Brand Content Batch → Outreach Strategy — with one starting click.
🧠 Making the Workforce Yours
8
Knowledge Base + Semantic Tags
Settings → URL Sources auto-indexes any URL into the KB. In the KB modal, assign Purpose Tags like [GRANT_GUIDELINES] or [TECHNICAL] — agents force-inject tagged items when running relevant steps, so the right docs are always in context. KB items are shown with their tags in the list view.
9
Long-Term Brain
Every memory is injected into every agent prompt, every time. Write here like you're briefing a new employee: "Always lead with user impact numbers," "Never use the word leverage," "The Community Pilot has 12 active users." The Mail Hub can auto-extract learnings from emails with the 🧲 Extract Learning button.
10
Snapshots — Rewind Anytime
Auto-snapshots fire before every pipeline run. If something goes wrong, go to Snapshots → Rewind to restore your entire workspace state — agents, KB, memory, custom workflows, departments. Last 10 checkpoints kept. Snapshot manually anytime from the Snapshots view.
🔭 Intelligence Hub
11
Five Intel Tools
Grants — live Grants.gov API for nonprofits and funded orgs. Competitor Pulse — structured competitive brief from any name or URL. Trend Spotter — paste raw customer voice (reviews, threads, tickets) and get structured signals. Lead Scanner — score and brief any prospect list. Compliance Check — review any document against your KB, brand rules, or mission.
⚡ Power Features
12
URL-Triggered Execution
Add ?run=weekly-review to the URL and that workflow fires automatically on load. Create desktop shortcuts for your Monday morning scan, weekly review, or social batch. Add &ctx=your+context to inject extra context.
13
🔧 Fix This — Emergency Dispatch
Click the red Fix This button in the toolbar. Describe any problem in plain English — "our grant language is too corporate," "the outreach email doesn't match our voice." The Orchestrator picks the right agent, executes a targeted fix, and returns a copyable result. One prompt, one click.
14
Team Packs & Multi-Workspace
Export a Team Pack (sidebar footer) to share your agent roster, departments, custom workflows, and memories — no API keys, no private content. Import into any workspace for instant setup. Add a new company with + Add Company in the top bar; each workspace is fully isolated.
15
Output Vault
Every pipeline, social batch, outreach draft, and fix lands in the Output Vault automatically. Search by keyword, filter by type. Your agents reference this to avoid duplicate work. Export the vault as JSON for records or sharing.
💡 Token tip: A 3-step pipeline uses ~1500 tokens. With Critic Loop enabled, add ~1 call per step. The free Groq tier gives ~500K tokens/day on llama-3.3-70b — enough for heavy daily use. Token count is always in the top bar.
