❓ How To Use ForceIQ
1
Configure Settings First
Before anything else, open ⚙️ Settings. Add your Groq API key (required). Fill in Company Profile — name, mission, founder name. Add your Voice & Tone. Without this, agents have no context and outputs will be generic.
2
Deploy Your Agents
Go to Hiring Hall. Click "+ Hire Agent" to add agents. Each agent needs a Role, Goal, and Backstory. The Goal is the most important — be specific about what this agent must accomplish. Agents run sequentially in pipelines, passing output to the next agent.
3
Run a Workflow
Go to Workflows → Templates. Click any template to load it into the pipeline. Or use the "Quick Deploy" on the War Room — type a task, select a workflow, hit Deploy. Each step runs in order: Agent 1 runs, Agent 2 reads Agent 1's output, and so on. Real sequential handoffs.
4
Build Social Content
Go to Social Engine. First configure your platforms in Settings → Social Platforms. Then describe a milestone or update, pick a tone, check which platforms to generate for, and hit Generate. You get platform-specific posts — LinkedIn gets a different format than Instagram.
5
Draft Outreach Emails
Go to Outreach Builder. Add your target audiences in Settings → Target Audiences first. Then select an audience, pick your goal (test invite, partnership, etc.), and generate. You get email drafts written in your voice, for that specific audience.
6
Index Your Knowledge Base
Go to Settings → URL Sources. Add your app portfolio URL, grant guidelines, brand pages. ForceIQ fetches and indexes them. Agents automatically reference these documents when relevant — they cite the source so you know where information came from.
7
Train Long-Term Memory
Go to Long-Term Brain. Add learnings like "Always mention the Community Pilot program in parent outreach" or "Grant applications should lead with user impact numbers." Every agent reads these before every task. This is what makes the workforce yours over time.
💡 Token tip: Each Groq call costs tokens. A full 3-agent pipeline typically uses 800-1500 tokens. The free Groq tier gives you ~500K tokens/day on llama-3.3-70b — more than enough for heavy use. Your token count is always visible in the top bar.
